# kh_reservation_ver2




## 주제
식당 예약 프로그램🍷



### 요구사항
1. 이름, 연락처, 인원, 날짜 등을 입력받는다.
2. 예약자 전화번호을 통해 예약 현황을 확인할 수 있다.
3. 예약자 이름을 통해 예약 현황을 확인할 수 있다. (동명이인 중복 출력 가능)
4. 연락처를 통해 예약 인원 / 일시를 변경할 수 있다.
5. 연락처를 통해 예약을 취소할 수 있다.



### 동작화면


"~~~~~@~~~ VineWine 예약 프로그램 ~~~@~~~~~"

5. 예약자 이름으로 예약 확인
